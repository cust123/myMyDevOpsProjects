DevOps Deployment Project

Cloud Infrastructure | CI/CD Pipelines | Real-World Practice

Project Overview

This DevOps deployment exercise is hosted on AWS EC2 and aims to simulate real-world infrastructure and deployment scenarios. It includes hosting a static HTML page, managing domain routing, and optionally integrating automation workflows.

Technologies Used

  • HTML5 and CSS3 for the web interface
  • AWS EC2 for hosting the web server
  • Nginx or Apache as the web server
  • Domain management and DNS routing
  • Optional: Git, CI/CD pipelines, SSL certificates

Next Steps

After setting up the site, explore integrating a CI/CD pipeline using GitHub Actions, implement SSL for secure connections, and use monitoring tools to track uptime and performance. Optionally, add Docker or Kubernetes for container orchestration practice.
